Stealing Steve

During my second year of University, I was part of a team consisting of 4 people to develop the mobile game ‘Stealing Steve’. Stealing Steve is an endless running game where the player tries to collect as many coins as possible whist avoiding the Police coming the other way. This is done be dodging either left or right into another lane.

As part of this project I took the roles of UI Developer and Head of Visuals, taking responsibility of the in-game navigation designs (Menus, Exit screen, Settings) as well as Icon and Title screen design. During this project I leant vital software development skills such as version control, Code collaboration and Agile (SCRUM) related tasks.
